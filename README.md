## Setup
To run this project, install it locally using npm:

* Clone this repository
$ git clone https://github.com/ewelina-slepko/sunny-or-rainy

* Go into the repository
$ cd sunny-or-rainy

* Install dependencies
$ npm install

* Run the app
$ npm start
